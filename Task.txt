Как должна работать синхронизация:

Для всех записей:
Id не переносится, а автоматически генерируется базой данной
Поле created_at переносится из sourceDB

1. На вход в программу подается id записи из таблицы report в sourceDB.
2. Необходимо перенести все данные связанные с этим отчётом (как дочерние, так и родительские)
3. Учесть тот факт, что id в sourceDB и targetDB будут разные. Использовать ключи для поиска нужных записей
